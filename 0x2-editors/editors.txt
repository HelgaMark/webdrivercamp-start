0. What is the command to open a file from within Emacs?
   - C-x C-f (Control + x, followed by Control + f)

1. What is the command to save a file?
   - C-x C-s (Control + x, followed by Control + s)

2. What is the command to cut an entire line?
   - C-k (Control + k)

3. What is the command to paste?
   - C-y (Control + y)

4. What is the command to undo?
   - C-/ or C-x u (Control + / or Control + x, followed by u)

5. What is the command to quit Emacs?
   - C-x C-c (Control + x, followed by Control + c)

Vi
7. What is the command to insert text before the cursor, after the cursor?
   - Before the cursor: i
   - After the cursor: a

8. What is the command to delete and cut the current line?
   - dd

9. What is the command to undo what you just did?
   - u

10. What is the command to quit vi even though latest changes have not been saved for this vi call?
    - :q!

11. What is the command to move the cursor to the start of the current line?
    - 0 (zero)

12. What is the command to move the cursor to the end of the line?
    - $
